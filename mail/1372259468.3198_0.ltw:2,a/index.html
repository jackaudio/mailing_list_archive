Hi James,

make sure you start jack as regular user (not root), something like su -l
username -c command in /etc/rc.local

uncomment in /etc/pam.d/su:
#session    required   pam_limits.so

(from
http://www.raspberrypi.org/phpBB3/viewtopic.php?f=38&t=33462&p=357866&hilit=rc.local+jack+rc.local#p357866)

hope that helps,
Thomas
