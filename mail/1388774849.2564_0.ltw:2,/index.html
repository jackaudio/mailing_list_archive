Kjetil Matheussen <[hidden]> wrote:

> So a while ago (still), I changed my .asoundrc file like this:
> 
> pcm.rawjack {
>     type jack
>     playback_ports {
>         0 system:playback_1
>         1 system:playback_2
>     }
>     capture_ports {
>         0 system:capture_1
>         1 system:capture_2
>     }
> }
> 
> pcm.jack {
>     type plug
>     slave { pcm "rawjack" }
>     hint {
>   description "JACK Audio Connection Kit"
>     }
> }
> 
> pcm.!default {
>     type plug
>     slave { pcm "rawjack" }
> }
> 

Brilliant ! This works for me, at least for watching a youtube video
while jack is up.
