Few days ago, we released JACK Audio Connection Kit for Android:
 - runs on latest android 4.4 platform
 - based on JACK2(jack-1.9.9.5)
 - support both ALSA and OpenSL ES driver
 - support emulator
 - code arranged. (android specific changes moved to /android folder)
 - fixed minor bugs.

You able to find source code and binaries at:
https://github.com/KimJeongYeon/jack2_android
(This project was retrieved from 'Samsung Open Source Release Center')

In this time, we provide binaries ready to run on emulator environment
with sound.
For more detail, please visit above link.
